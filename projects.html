Those Who Came After

MAY 2019

Project Duration: 3 Weeks

Keywords: Text-adventure, History, Game

Live Preview

Source

This app is based on the popular text adventure games made popular in the early 80's. You play as a native of the island of Hispanola looking to escape your enslavement.

Technologies Used:

  • Unity 2D

  • C#

  • Visual Studio Code

  • PhotoShop

  • HTML5

  • CSS3

  • Audacity

My Role

In this project I oversaw all aspects of development. My intial interest in the topic came from a course I was taking my senior year at New York University, which focused on literature based in the Carribean.

Project Difficulties

After researching the ways in which I could develop a text-adventure game using Unity, I realized the amount of items I would ultimately need to implement was a lot more than I had anticipated. I was essentially working with over a hundred items and no way to really organize them.

My Solution

Through some in depth organization, I was able to develop a chart with all items as well as prompts that would lead to: obtaining the items, directional cues, and blocked passages that would prevent the player from progressing. The process became much quicker and simpler, and allowed me to focus most of my time on the story.

Notable Features

The simplicity of this app is definitely something that reminds me of the classic text-adventure games I would play growing up. Despite there being no images, I believe the audio cues and plot help bring this story to life.

Metamorfosis

MAY 2019

Project Duration: 1 Month

Keywords: VR, Womanhood, Art

Live Preview

Source

This app was developed for a New York University course which focused on Digital Media. For this project I developed the idea for a VR art piece, with a focus on aspects of woman hood.

Technologies Used:

  • Unity 3D

  • C#

  • Visual Studio Code

  • PhotoShop

  • HTML5

  • CSS3

  • Audacity

My Role

In this project I oversaw all aspects of development. Being that I am also an artist, I decided to create original digital srt pieces on topics in today's world. The three topics I focused on were 1) The Emotional Being 2)Queen Bee Leading, Whose to Follow and 3) Accepting the Unknown

Project Difficulties

Creating the actual VR space was by far the most difficult. This was my first attempt at using Unity's 3D engine, so I had to learn how to navigate the actual interface. Once I was able to successfully program the engines cameras, I was more easily able to develop the space to fit my needs.

My Solution

My prior experience with Unity's 2D engine aided me in understanding the basic functionalities of several 3D elements. I was able to adapt my knowledge of Unity's 2D camera to create a 3D camera that captured the space in a way that focused primarily on the art.

Notable Features

The atmospheric music in the background is something I added at the very end and is something I believe really pulled the experience together. The moving art pieces add movement to the messages I wanted to come across.

In the Stars

MAY 2019

Project Duration: 2 Weeks

Keywords: Poetry, Astrology, Art

Live Preview

Source

This app was developed with my interest in astrology, and the ways in which the stars connect to each horoscope sign. All SVG's were created my me, individually using Adobe Illustrator.

Technologies Used:

  • JavaScript

  • HTML5

  • CSS3

  • PhotoShop

  • Illustrator

My Role

In this project I oversaw all aspects of development. My goal was to create an art piece that was simple yet emotional. I developed each SVG as well as the poem that went with each horoscope.

Project Difficulties

Creating each SVG was by far the most difficult aspect of this project. After developing each image, I had to animate each using Illustrator, only to then connect each piece through code using JavaScript.

My Solution

Working in parts, while simple, served as the most effienct way to get each drawing done on time. First starting with the line work, and then focusing on each glowing orb allowed me to work quickly and efficiently.

Notable Features

For this piece I wanted to make sure that the viewer took their time to look through each page. There are no instructions, as the goal is for the viewer to explore at their own pace.

I'm Innocent

DECEMBER 2019

Project Duration: 1 Month

Keywords: Mini-Games, Prison, Non-Fiction

Live Preview

Source

This app was created as a group project with the theme of the U.S. prison system. My collegues and I created 4 different mini-games with the main gola of creating an experience in which the player would be required to pass each mini-game in order to escape a prison.

Technologies Used:

  • JavaScript

  • HTML5

  • CSS3

  • PhotoShop

  • Audacity

  • MAMP

  • P5

My Role

In this project I oversaw the development of the mini-game "Search and Escape"

Project Difficulties

After coming up with the idea of creating a search and escape game, I struggled with figuring out with how to create enemeies that would cause damage, and also move around the map without leaving the map and causing the game to crash.

My Solution

Through Perlin Noise and in depth testing, I was able to create a map with values that would limit the enemies to a specific space of the map. The do not crash into each other, but can cause damage to the player if they bump into each other.

Notable Features

The map was created using an array in which number represent certain tiles that I designed. There are tiles such as: door tiles, floor tiles, wall tiles, etc.

Circus of Values

NOVEMBER 2018

Project Duration: 1 Week

Keywords: BioShock, Wack-a-Mole, Clownn

Live Preview

Source

This app was based on the classic game Wack-a-Mole. As a fan of the BioShock game series, I decided to create a Wack-aMole inspired by the famous clown themed vending machine, better known as the Circus of Values.

Technologies Used:

  • JavaScript

  • HTML5

  • CSS3

  • PhotoShop

  • Audacity

  • MAMP

  • P5

My Role

In this project I oversaw all aspects of development, including art, sounds, and game mechanics. With the assistance of the P5 library, I was more easily able to develop a canvas in which the actual game would live on.

Project Difficulties

The most difficult part of this project was coming up with the flipping mechanic for all of the tiles. I had to figure out a timing system that would alternate between tiles randomly while also differentiating between each tile.

My Solution

By randomizing the time to flip as well as the type of tile that would present itself, I was able to develop a game space in which each tile was effectively randomized.

Notable Features

Similar to the actual BioShock game, my game has various audio clips of the quirky vending machine clown. This definitely brings the game to life.

Shoot Em' All

October 2018

Project Duration: 1 Week

Keywords: Birds, Shooting, Hunter

Live Preview

Source

This app was developed with the P5 library, and was inspired at by the most recent release of Duck Hunt. I decided to create a bird shooting game that had some sort of timed element, as well as an enviornment that made the player feel like an actual bird hunter.

Technologies Used:

  • JavaScript

  • HTML5

  • CSS3

  • PhotoShop

  • Audacity

  • MAMP

  • P5

My Role

For this app I developed the concept as well as the various sounds you hear when you play. I wanted to create a simple game that didn't need to use too much code, but played very sophistically.

Project Difficulties

The trickiest part of this app was probably the actual birds. I wasn't initially sure how to make the birds spit out from one side. I had to make sure that each bird would spawn at a random X and Y coordinate and disappear once they reached the other side.

My Solution

Using a for loop and an array of images of birds I picked out, I was able to randomize the X and Y coordinates of each bird based on the canvas location, as well as spit out random birds to add an extra element to my app.

Notable Features

As an added difficuly, there is a slider that either slows down or speeds up the rate at which the birds fly from one side to the other.

Space Pong

September 2018

Project Duration: 1 Week

Keywords: Space, Pong, Astronaut

Live Preview

Source

This app was was created with my interest of space in mind. I wanted to create a simple Pong game with a deep atmospheric tone. Using the right sounds was very important to me as I wanted to make sure the player felt the stress of playing the game and having to try to save each astronaut.

Technologies Used:

  • JavaScript

  • HTML5

  • CSS3

  • PhotoShop

  • Audacity

  • MAMP

  • P5

My Role

As the only developer of this app, I was responsible for all aspects of this game. While a lot of my time went into the actual coding of this game, I also wanted to make sure this app had an atmospheric element to it.

Project Difficulties

The most difficult part of this app was probably the paddle of the game. It was difficult to create a smooth movement to the paddle whenever the player moved from one side to another.

My Solution

To fix the issue with the paddle, I used the constrain feature to limit the overall speed and make it so that the paddle does not move too quickly. I also did this with the ball in order for it to not move too quickly.

Notable Features

The audio in this app is probably my favorite part as it really feels as if you are in space.
